# cema-app
An advanced, multi-role web application built on ASP.NET Core MVC (8.0), Entity Framework Core, and SQL Server. CemaApp is designed to manage cinema schedules, seat bookings, administrative reporting, and concurrency-safe, real-time ticket transactions.
